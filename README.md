## What is kiwi?

Kiwi is a simple, educational key–value store project in Go.

> Why “Kiwi”?
> A playful nod to “KV” (key–value). Spoken in English (“kay-vee”), it sounds a bit like the german fruit “Kiwi” — and thus a friendly fruit mascot was born.
